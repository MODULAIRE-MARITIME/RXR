Mise à jour firmware

Cette page permet d’installer ou mettre à jour le firmware de votre Drum Sampler basé sur ESP32-S3 Plus.

Utilisez Chrome ou Edge sur Mac ou PC, puis branchez le module en USB.

  1. Branchez le module en USB.
  2. Si nécessaire, mettez-le en mode bootloader (maintenir BOOT puis appuyer sur RESET, puis relâcher BOOT).
  3. Cliquez sur le bouton ci-dessous.
  4. Sélectionnez le port USB correspondant au module.
  5. Laissez l’installation se terminer sans débrancher le câble.
Si rien ne se passe au clic, essayez un autre câble USB data ou relancez le module en mode bootloader.

Cette page utilise ESP Web Tools, qui permet le flash firmware directement depuis le navigateur via Web Serial. [web:294][web:328]
