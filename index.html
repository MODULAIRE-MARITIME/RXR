MM RXR Firmware Update

This page lets you install or update the firmware of your MM RXR module.

Please use Chrome or Edge on Mac or PC, then connect the module over USB.

  1. Connect the module to your computer with a USB data cable.
  2. If needed, put the module into bootloader mode by holding BOOT, briefly pressing RESET, then releasing BOOT.
  3. Click the install button below.
  4. Select the USB port corresponding to the module.
  5. Wait for the installation to complete before disconnecting the cable.
If nothing happens when you click the button, try another USB data cable or restart the module in bootloader mode.

This page uses ESP Web Tools to flash firmware directly from the browser.
